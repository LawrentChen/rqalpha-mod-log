## Enable/Disable log Mod

```bash
# Disable log Mod
$ rqalpha mod disable log

# Enable log Mod
$ rqalpha mod enable log
```

## Mod Config

Configs of log Mod are as followed.

```python
{
    # Specify the log output file
    "log_path": "./log.txt",
    # Python open() method to read/write file
    "log_mode": "a",
}
```
